## Suspense란
- 컴포넌트에서 필요한 데이터가 아직 준비가 되지 않았다고 리액트에게 알려주는 기능

## Suspnse 동작 원리
- Suspense도 동시성 렌더링과 결을 같이 한다.

## Suspense를 사용하기
- React.lazy
  - 무조건?
- Data Fetching
- 커스텀 사용법

## Suspense 사용시 주의사항
- tanstack query suspense
- concurrent rendering
